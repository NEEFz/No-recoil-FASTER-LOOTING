วิธีใช้งาน loot fast ต้องใช้ 
locate เพื่อตั้งค่าที่ x,y ที่ a+l fast หากใช้ 1920x1080 จะไม่มีปัญหา

ต่อมาการใช้ recoil 
คุณสามารถตั้งค่าได้ด้วยตนเอง ผมไม่สามารถตั้งค่าให้พอดีกับทุกคนได้ ส่วนตั้งค่าผมใช้ dpi 3200 ส่วนตั้งค่าตามรูปเลย

*FASTER-LOOTING
To use loot fast, you need to use `locate` to set the x,y coordinates for `a+l fast`. This should work fine with 1920x1080 resolution.
