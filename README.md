# No-recoil-FASTER-LOOTING
It's a code that helps make the game more fun.
This is code I wrote with some AI-assisted modifications. You can use it for free and provide feedback using AutoHotkey.
*Do not use for commercial purposes.*

=====================loot fast===================================

วิธีใช้งาน loot fast ต้องใช้ (กด mouse 5 button หรือสามารถตั้งปุ่มอื่นได้)
locate เพื่อตั้งค่าที่ x,y ที่ a+l fast หากใช้ 1920x1080 จะไม่มีปัญหา ปรับจูนได้ที่บรรทัด 130-135

=====================No-recoil===================================

ต่อมาการใช้ recoil (T เปิด/ปิด การใช้งาน , U เลือกปืนที่ต้องการ , Caplock เปลี่ยนเป็นสโคปซูม X4)
คุณสามารถตั้งค่าได้ด้วยตนเอง ผมไม่สามารถตั้งค่าให้พอดีกับทุกคนได้ ส่วนตั้งค่าผมใช้ dpi 3200 ส่วนตั้งค่าตามรูปเลย
การตั้งค่าหรือการจูนให้เข้ากับความเร็วเมาส์ สามารถปรับได้ที่ในไฟล์ ถ้าปืนยังดีดขึ้นหรือปืนดีดลง ให้เพิ่มหรือลดเลข
จะอยู่ที่บรรทัดที่ 17-20 ใส่
สามารถใช้ได้กับสโคป X4 ปรับจูนได้ที่บรรทัดที่ 23-24

======================================================loot fast=========================================================================================

*FASTER-LOOTING (Press 5 mouse buttons or you can assign other buttons.)
To use loot fast, you need to use `locate` to set the x,y coordinates for `a+l fast`. This works fine with 1920x1080 resolution. Adjustments can be made on lines 130-135.

======================================================No-recoil========================================================================================

*No-recoil (T Enable/Disable, U Select desired weapon, Caplock changes to 4x zoom scope)
Next, regarding recoil:
You can set this manually. I can't optimize it for everyone. I use 3200 DPI; see the image for the settings.
The settings to adjust for mouse speed can be done in the file. If the gun still recoils, increase or decrease the numbers.
These are located on lines 17-20.
This works with X4 scopes; adjust on lines 23-24.
